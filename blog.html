Discuss the scope of var, let, and const

Let

Let is a block scope which once declare in a scope cannot redeclare but reinitialize the value from the same scope.

const

const is a block scope which once declare in cannot redeclare or cannot reinitialize from anywhere.

var

var is a not a block scope rather it is a global scope means it can be reinitialize or redeclare from anywhere which once declare in a scope cannot redeclare but reinitialize the value.
